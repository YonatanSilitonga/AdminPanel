## 🚀 CARA MENGGUNAKAN

### 📖 Baca Dokumentasi:

1. **Mulai dari:** `docs/README.md`
2. **Lihat index:** `docs/INDEX.md`
3. **Ikuti panduan:** `docs/NAVIGATION_GUIDE.md` (sesuai role Anda)

### 📊 Lihat Diagram:

**Opsi 1: Online (Tercepat)**
1. Buka file `.puml` dengan text editor
2. Copy semua isi file
3. Buka: https://www.planttext.com/
4. Paste & klik "Refresh"

**Opsi 2: VSCode Extension**
1. Install extension: "PlantUML" by jebbs
2. Buka file `.puml`
3. Tekan `Alt+D` untuk preview

**Opsi 3: Command Line**
```bash
java -jar plantuml.jar docs/diagrams/sitemap-monitoring-settings.puml
```
